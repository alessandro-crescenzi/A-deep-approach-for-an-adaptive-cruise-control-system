1. Fare pull del repository

2. Scaricare i pesi dalla cartella condivisa:
	- andare su 'CV Project weights/YOLACT'
	- scaricare il file 'yolact_base_14_5565.pth'
	- mettere il file nel percorso del repository 'computer_vision_project/YOLACT/weights'

2. Per eseguire la rete di instance segmentation:

	- Di una sola immagine:
		
		python eval.py --trained_model=weights/yolact_base_14_5565.pth --score_threshold=0.15 --top_k=15 --image=path/to/image
		
	- Di una sola immagine e salvarla su un file
	
		python eval.py --trained_model=weights/yolact_base_14_5565.pth --score_threshold=0.15 --top_k=15 --image=input_image.png:output_image.png

	- Di tutte le immagini presenti su una cartella

		python eval.py --trained_model=weights/yolact_base_14_5565.pth --score_threshold=0.15 --top_k=15 --images=path/to/input/folder:path/to/output/folder

	- Process a video and save it to another file. This uses the same pipeline as the ones above now, so it's fast!
	
		python eval.py --trained_model=weights/yolact_base_14_5565.pth --score_threshold=0.15 --top_k=15 --video_multiframe=4 --video=input_video.mp4:output_video.mp4
		
		(In questo caso potete regolare il parametro video_multiframe in base alla memoria della GPU che utilizzate)
